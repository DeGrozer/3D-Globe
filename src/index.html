About Globe 3D Render

An interactive 3D globe visualization built with Three.js. Click on any country to learn more about it.

Controls

  • Drag - Rotate the globe
  • Scroll - Zoom in/out
  • Click - Select a country

Technologies

  • Three.js for 3D rendering
  • Wikipedia API for country info
  • REST Countries API for data
